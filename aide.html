✅ Heures pré-remplies à 7.5h par jour
✅ Gestion de plusieurs chantiers par ouvrier

✅ Sélection du conducteur pour chaque jour

✅ Calcul automatique des totaux par chantier
✅ Ajout d'observations par ouvrier
✅ Marquage des ouvriers intérimaires
✅ Sélection/création de chantiers
✅ Ajout/création d'ouvriers

Installation

Aucune installation nécessaire ! Il suffit d'ouvrir le fichier index.html dans un navigateur web moderne (Chrome, Firefox, Edge, Safari).

Utilisation

1 Configuration de la semaine

  1. Sélectionner la semaine : Utilisez le sélecteur de semaine en haut de la page
  2. Désigner le chef de chantier : Sélectionnez le chef de chantier dans la liste déroulante
  3. Choisir les conducteurs : Pour chaque jour (lundi à vendredi), sélectionnez qui conduit (par défaut le chef de chantier)
  4. Ajouter des ouvriers : Cliquez sur "Ajouter un ouvrier" pour :
    • Sélectionner un ouvrier préenregistré
    • Ou créer un nouvel ouvrier (prénom et nom)

2 Saisie des heures

Pour chaque ouvrier :

  1. Sélectionner le chantier : Cliquez sur le bouton "Sélectionner un chantier" pour :
    • Choisir un chantier prédéfini (Brouage, Château d'Oléron, Consac, etc.)
    • Ou créer un nouveau chantier
  2. Modifier les heures si elles diffèrent de 7.5h (pré-remplies par défaut pour le premier chantier)
  3. Ajouter un chantier si l'ouvrier a travaillé sur plusieurs chantiers dans la semaine
  4. Ajouter une observation (optionnel) : Cliquez sur "Ajouter une observation" pour saisir des remarques
  5. Marquer comme intérimaire (si applicable) : Cliquez sur le bouton "Intérimaire" pour activer/désactiver le statut
  6. Les totaux se calculent automatiquement

3 Cas d'usage courants

📌 Ouvrier sur un seul chantier toute la semaine

  • Sélectionnez simplement le nom du chantier
  • Les heures sont déjà pré-remplies à 7.5h
  • Modifiez uniquement si nécessaire

📌 Ouvrier sur plusieurs chantiers

  • Sélectionnez le premier chantier avec ses heures
  • Cliquez sur "+ Ajouter un chantier"
  • Sélectionnez le deuxième chantier avec ses heures
  • Mettez à 0 les jours non travaillés sur chaque chantier

📌 Ouvrier absent un jour

  • Mettez 0 dans la case du jour d'absence

4 Export et impression

  1. Imprimer : Cliquez sur le bouton "Imprimer / PDF" en haut à droite
  2. Enregistrer en PDF : Dans la fenêtre d'impression, choisissez "Enregistrer au format PDF"
  3. Le rapport inclut :
    • La période de la semaine
    • Le chef de chantier
    • Les conducteurs pour chaque jour
    • Le détail par ouvrier et chantier avec les heures
    • Les observations de chaque ouvrier
    • Le statut intérimaire (si applicable)
    • Les tableaux récapitulatifs (totaux par ouvrier et par chantier)

Remarques importantes

⚠️ Pas de sauvegarde automatique : Les données sont perdues si vous fermez la page. Pensez à imprimer ou exporter en PDF avant de fermer.

💡 Navigateur moderne requis : Fonctionne avec Chrome, Firefox, Edge, Safari (versions récentes)

📱 Responsive : Fonctionne sur ordinateur, tablette et mobile

Support

Pour toute question ou problème, vérifiez que :

  • Vous utilisez un navigateur moderne et à jour
  • JavaScript est activé dans votre navigateur
  • Vous avez une connexion internet (pour charger TailwindCSS et Lucide icons)
